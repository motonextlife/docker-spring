詳解

主にシステム全体のバイナリファイル入出力関連で使用する、入力したファイルが システム上で許容できない状態のときに発する例外。

  • この例外を発する状況としては、入力したバイナリファイルがデータ量やデータ形式などで システムの要件を満たさなかった場合になる。
  • この例外は、システムの全域で用いる。
  • また、ファイルサーバーから取得してきたバイナリデータが要件を満たしていなかったり、 改ざんがされていた場合に発する。
