void com.springproject.dockerspring.JoinTests.FileIO.PdfProcess_Test.PdfProcess_MakePdf_Test.PDF出力機能の動作確認と生成ファイルの存在確認 ( ) throws IOException, ParseException

PDFの出力処理において、対象のPDFファイルが正常に出力されていることを確認する。

覚え書き
このテストでは、「出力したPDFファイルの存在確認」のみを行う。理由としては、jUnitの自動 テストでは、PDFのデザインの判定はできないかつ、デザインは今後頻繁に変わる不定値の為、自動で ファイルの中身の正当性を確認することが困難なため。デザインは、実際に出力して人の目で確かめる。
